xTimerStop 是关闭一个软件定时器，xTimerCreate 是创建一个软件定时器，xTimerStart 是启动软件定时器，xTimerReset 是重置软件定时器，xTimerChangePeriod 是修改软件定时器的周期

 软件定时器的 API

	![[Pasted image 20250629151314.png]]

	形参为定时器名称，定时器周期（单位为tick），uxAutoReload用于指定定时器是否自动重新加载。%% 如果设为非零值，定时器到期后会自动重新加载初始值并继续计时；如果设为零，定时器只会触发一次 %%

	 pvTimerID这是一个指向空类型的常量指针，可以传递一个自定义的标识符给定时器，通常用于在回调函数中区分不同的定时器。 pxCallbackFunction这是一个函数指针，指向定时器到期时要调用的回调函数。当定时器的定时周期结束时，就会调用这个函数。
