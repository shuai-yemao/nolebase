==cmsis-rtos api 中的原生任务创建函数==

pxtaskcode 表示指向任务函数的指针

![[Pasted image 20250627153156.png]]![[Pasted image 20250627153243.png]]

任务函数结构体，包括任务名称，任务堆栈大小，和任务优先级

pvparameters 是传给任务函数的形参

TCB_t 结构指针是任务控制块，该结构体包含了一个任务的所有信息

	![[Pasted image 20250627154038.png]]

	pxtopofstack是栈顶指针，指向任务的栈顶

	xstatelistItem是状态链表，阻塞表，挂起表，运行表等；

	xeventlistItem是事件链表

	pxstack为任务分配堆栈地址空间，可以用来检测堆栈是否溢出
