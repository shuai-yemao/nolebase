## 2. 逐点精讲

### 知识点 1：局部焊接局部测试

#### 实际意义

快速定位硬件问题，提高工作效率

#### 常见误区

将 PCB 元件一次性全部焊接完，测试 PCB 板时无法快速定位问题

#### 核心逻辑/原理

1. 从电源部分开始
2. 先焊接有自检功能的芯片

#### 关键公式/结论

1. 焊完电源部分需要使用电压表测试电路是否短接或者断路
2. 使用示波器测试输出电压，并记录其电压波形
3. 注意人体静电导致器件击穿，ESD 装备可以防止

### 知识点 2：二极管

#### 实际意义

 二极管是电子文明的起源

#### 应用场景

1. 防止电源反接
2. 防止电源倒灌
3. 整流，交流电转直流电
4. 用于照明发光二极管
5. 用于感光的光敏二极管
6. 用于稳压的稳压二极管
7. 等等

#### 辅助图示

1. PN 结 ![[Pasted image 20260310161606.png]]
2. 反电源反接电路 ![[Pasted image 20260310161730.png]]
3. 防止电流倒灌 ![[Pasted image 20260310161810.png]]
4. 整流桥 ![[Pasted image 20260310161210.png]]

#### 核心逻辑/原理

 单向导电性，电流只能从二极管正极流向负极（正向导通），反向截止，N 型半导体带负电（自由电子为多子，空穴为勺子），P 型半导体带正电（空穴为多子），由于向右的电场（正向）会让电子受到向左的电场力移动到 P 型半导体中形成电流，反之向左的电场（反向）会让电子待在 N 型半导体中，无法形成电流

#### 关键公式/结论

1. 当电流超过二极管的最大承载电流时二极管会被损坏
2. 当二极管反向电压超过最大反向电压二极管被击穿（热击穿）
3. 二极管两极电压超过最小工作电压时二极管导通

### 知识点 3：三极管

#### 应用场景

1. 逻辑门电路
2. 放大电路
3. 震荡电路
4. 开关电路

#### 辅助图示

1. NPN 三极管 ![[Pasted image 20260310164301.png]]
2. PNP 三极管 ![[Pasted image 20260310165928.png]]
3. 三极管开关电路 ![[Pasted image 20260310171025.png]]![[Pasted image 20260310171351.png]]

#### 通俗人话解释

一根小水管来控制大水管的出水，小水管流量达到一定量后就会打开大水管的部分阀门，小水管水越多，大水管的水也越多

#### 核心逻辑/原理

1. 发射区向基区发射电子:
	 电源 Ub 经过电阻 Rb 加在发射结上，发射结正偏，发射区的多数载流子 (自由电子）不断地越过发射结进入基区，形成发射极电流 Ie。同时基区多数载流子也向发射区扩散，但由于多数载流子浓度远低于发射区载流子浓度，可以不考虑这个电流，因此可以认为发射结主要是电子流。

2. 基区中电子的扩散与复合
	 电子进入基区后，先在靠近发射结的附近密集，渐渐形成电子浓度差，在浓度差的作用下，促使电子流在基区中向集电结扩散，被集电结电场拉入集电区形成集电极电流 Ic。也有很小一部分电子（因为基区很薄）与基区的空穴复合，扩散的电子流与复合电子流之比例决定了三极管的放大能力。

3. 集电区收集电子
	 由于集电结外加反向电压很大，这个反向电压产生的电场力将阻止集电区电子向基区扩散，同时将扩散到集电结附近的电子拉入集电区从而形成集电极主电流 Icn。另外集电区的少数载流子（空穴）也会产生漂移运动，流向基区形成反向饱和电流，用 Icbo 来表示，其数值很小，但对温度却异常敏感

#### 关键公式/结论

1. B 为基极，C 为集电极，E 为集电极
2. NPN 导通条件 Ube>0.7 v(锗 0.3 v，硅 0.7 v)，Ub>Uc；PNP 导通条件 Ueb>0.7 v ,Uc>Ub
3. 线性放大作用，Ic = βIb 即发射极电流时基极电流的β倍
4. 小电流控制大电流
5. 三极管的状态分为饱和，放大，截止
6. Ie=Ib+Ic

## 4. Q&A

### Q 1：为什么电路板要进行局部焊接局部测试？

 A 1: 高效完成硬件设备检测，排查硬件问题，PCB 不能正常工作时候可以快速定位问题

### Q 2：什么是半导体二极管？它的工作原理是什么？

A 2: 由 N 型半导体和 P 型半导体组成的 PN 结叫半导体二极管，其工作原理为 N 型半导体带负电（自由电子为多子，空穴为勺子），P 型半导体带正电（空穴为多子），由于向右的电场（正向）会让电子受到向左的电场力移动到 P 型半导体中形成电流，反之向左的电场（反向）会让电子待在 N 型半导体中，无法形成电流，总结为正向导通，反向截止

### Q 3：两个二极管 P 区相连能否形成一个三极管？

A 3: 不能，主要原因还是基区厚度的问题，就是说载流子有平均迁移长度，平均迁移长度就可以理解为是载流子在不同的区迁移的平均值（比如电子在 P 区），超过这个长度多数载流子会被复合，比如电子在 P 区只能移动很短的距离，超过这个距离电子就会被 P 区空穴复合。如果把两个二极管 P 端相连，则相当于 npn 管子基极 P 区很长，电子无法穿越 P 区，因此不能形成三极管。

### Q 4：解释 NPN 和 PNP 晶体管的区别及其应用，原理图中如何判断他们的符号？

A 4:

1. 发射极为 N 型半导体、基极为 P 型半导体、集电极为 N 型半导体的晶体管称为“NPN 型晶体管”。其 NPN 电流从集电极电流和基极电流流向发射极，原理图中箭头朝外的引脚是发射极 E，三极管为 NPN
2. 发射极为 P 型半导体、基极为 N 型半导体、集电极为 P 型半导体的晶体管称为“PNP 型晶体管”。其 PNP 电流从发射极流向基极和集电极，原理图中箭头朝内的引脚是集电极，三极管为 PNP
