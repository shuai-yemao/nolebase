# 设备一：手环

1. 心率检测模块心率检测（心率监测传感器）
2. 提醒吃药模块使用蜂鸣器发出一段提示音，提醒吃药
3. 跌倒检测模块使用 ADXL345 模块检测老人是否跌倒
4. 远程定位功能移远 EG800K 模块 GPS 定位 4G Cat 联网
5. 一键报警模块佩戴者按下按钮，立即发送最近心率与最近定位到服务器，服务器通过微信小程序向用户报警
6. 电源管理模块 设计电源管理模块保存

# 设备二：家庭环境检测

1. 燃气检测模块使用一氧化碳模块检测浓度，超出阈值使用蜂鸣器报警
2. 使用重量传感器检测是否服药使用压力传感器检测是否服药
3. ASR-PRO 模块语音控制设备使用 ASR-PRO 模块控制灯与家电（继电器代替）
4. 联网传输传感器数据使用 ESP01s 模块联网
5. 声光报警功能蜂鸣器，警示灯 RGB 模块
6. 测体温功能（元器件暂未到货） 使用红外测温模块，非接触式测温

# 微信小程序

1. 登录注册
2. 显示定位
3. 告警信息
4. 提醒吃药
5. 告警历史查询
