二叉树的递归遍历

-**先序遍历**：A → B → D → E → C → F`  
    （根优先访问，逐层向左延伸）
- **中序遍历**：`D → B → E → A → F → C`  
    （左子树完全遍历后访问根，再遍历右子树）
- **后序遍历**：`D → E → B → F → C → A`  
    （子树完全遍历后才访问根）![[Pasted image 20250827204928.png]]

层序遍历
