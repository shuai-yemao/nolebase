### 硬件过零比较原理

![[Pasted image 20250710094107.png]]

将正弦波转换成方波是第一步，通过电压互感器（变压器）将电压缩小，将输出的方波输入到定时器输入捕获，单片机测出频率和相位点，在中断中累加来获取实时的正弦波相位

### 软件过零比较原理

使用 [[ADC]] 借助电压测量模块（变压器）实测交流电压，在实际值的过零处，将状态位置一并且开始在中断中计数，等到下一个上升沿，状态位置零，结束计数，由此算出频率

![[Pasted image 20250710105304.png]]

首先用 [[ADC]] 通道对实际波形采样，实际波形过零处将鉴相器输出置一，在控制器内部虚拟波形上升沿过零处将鉴相器输出置零，输出为两个波形的相位差，经过滤波转为一个数值并且能转为相位偏差，给 pi 控制反馈，完成锁相

### DQ 坐标系

![[Pasted image 20250710110551.png]]

在稳定条件下，交流电压矢量与 d 轴，q 轴的夹角恒定，该矢量在两轴的分量为直流量，将对交流量的控制转化为对直流量的控制

#### DQ 锁相

![[Pasted image 20250710111114.png]]

调节 dp 坐标系旋转的转速与电网的电压旋转矢量 V 同步来实现锁相，此时 V 的 q 轴分量为 0

![[Pasted image 20250710111354.png]]

pi 的输出结果为 dq 坐标系旋转角速度，如何去改表角速度呢？修改相位角，改变频率，进而累加改变相位

### PLL 锁相环

基于 DSOGI PLL 锁相环的 SPWM 或 SVPWM 三电平三相逆变器控制代码实现主要包括以下几个步骤：

1. 初始化参数：设置逆变器的相关参数，如载波频率、调制比等。
2. 采集电网电压：通过传感器采集电网电压信号，为 DSOGI PLL 锁相环提供输入。
3. DSOGI PLL 锁相环运算：根据采集的电网电压信号，通过 DSOGI PLL 锁相环运算得到电网电压的相位信息。
4. SPWM 或 SVPWM 算法运算：根据得到的相位信息，通过 SPWM 或 SVPWM 算法运算生成 [[PWM]] 波形。
5. 输出控制信号：将生成的 [[PWM]] 波形转换为物理控制信号，驱动逆变器输出相应的电压。
