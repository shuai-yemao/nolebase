C++ 中的 vector 是一种序列容器，它允许你在运行时动态地插入和删除元素。

vector 是基于数组的数据结构，但它可以自动管理内存，这意味着你不需要手动分配和释放内存。

**基本特性:**

	- **动态大小**：`vector` 的大小可以根据需要自动增长和缩小。

	- **连续存储**：`vector` 中的元素在内存中是连续存储的，这使得访问元素非常快速。

	- **可迭代**：`vector` 可以被迭代，你可以使用循环（如 `for` 循环）来访问它的元素。

	- **元素类型**：`vector` 可以存储任何类型的元素，包括内置类型、对象、指针等。

**使用场景：**

	- 当你需要一个可以动态增长和缩小的数组时。

	- 当你需要频繁地在序列的末尾添加或移除元素时。

	- 当你需要一个可以高效随机访问元素的容器时。

___

数组和 vector 的比较

	1. **数组**

	    - **静态固定大小**：数组的长度在编译时确定，运行时无法修改1。例如，`int arr[5]`一旦定义，大小永久固定。

	    - **内存分配**：普通数组（非动态分配）通常在栈上分配，内存占用固定且连续。全局或静态数组则分配在静态存储区1。

	    - **手动管理**：需手动分配（`new[]`）和释放（`delete[]`）动态数组，否则会导致内存泄漏5。

	2. **`std::vector`**

	    - **动态可变大小**：通过`push_back`、`resize`等操作可在运行时动态调整大小，内部使用堆内存实现1。例如，`vec.resize(10)`可扩展或收缩容器。

	    - **自动内存管理**：`vector`在析构时自动释放内存，避免内存泄漏5。若需释放多余容量，可调用`shrink_to_fit()`8。

	    - **扩容策略**：当元素数量超过当前容量时，`vector`通常以指数方式扩容（如 2 倍），均摊时间复杂度为 O (1)78。预分配内存（`reserve`）可减少重新分配次数5。

##### 创建 vector

要使用 vector，首先需要包含 <vector> 头文件：

#include <vector>

创建一个 vector 可以像创建其他变量一样简单：

std::vector<int> myVector; // 创建一个存储整数的空 vector

这将创建一个空的整数向量,也可以在创建时指定初始大小和初始值：

std::vector<int> myVector(5); // 创建一个包含 5 个整数的 vector，每个值都为默认值（0）

std::vector<int> myVector(5, 10); // 创建一个包含 5 个整数的 vector，每个值都为 10

或：

std::vector<int> vec; // 默认初始化一个空的 vector

std::vector<int> vec2 = {1, 2, 3, 4}; // 初始化一个包含元素的 vector
