#### 参数传递

第一种参数传递是传递值，传固定字节时适用，不能传递不定字节（在 C 语言中的解决方法是传递指针），在 c++ 中所有参数传递尽量传递引用

	![[Pasted image 20250903195419.png]]

![[Pasted image 20250903200047.png]] 考虑到引用也是指针的一种形式，在传递过程中根据情况使用 const 修饰，防止函数中改变传递值

	![[Pasted image 20250903200302.png]]

#### 返回值

![[Pasted image 20250903200411.png]]

在 c++ 中尽量返回引用（为什么？）只有外部函数不调用且返回值时使用返回引用类型，这样调用得到的数值不会因为且返回值为 local 变量（函数运行结束，内存释放了），导致数值为乱码
