继承允许我们依据另一个类来定义一个类，方便快速创建一个类，提高执行效率

定新建的类继承了一个已有的类的成员即可。这个已有的类称为**基类**，新建的类称为**派生类**。

```
// 基类
class Animal {
    // eat() 函数
    // sleep() 函数
};

//派生类
class Dog : public Animal {
    // bark() 函数
};
```

一个类可以派生多个类，定义一个派生类，可以使用类派生列表来指定基类（类派生列表？）

类派生列表以一个或多个基类命名，形式如下：

class derived-class: access-specifier base-class

访问修饰符 access-specifier 是 **public、protected** 或 **private** 其中的一个，base-class 是之前定义过的某个类的名称，默认 access-specifier 为 private

#### 多继承

多继承即一个子类可以有多个父类，它继承了多个父类的特性。

```
class <派生类名>:<继承方式1><基类名1>,<继承方式2><基类名2>,…
{
<派生类类体>
};


```
