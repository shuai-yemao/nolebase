## 构造（出生）

标准 Init 设计模式

```
typedef struct {
	uint8_t initialized;
	void * resource;
	uint32_t config;
	
}Module_t;

int Module_Init(Modult_t * module, uint32_t config)
{
	if (module->initialized) return -1; // 防止重复初始化
	
	module->resource  = malloc(1024); // 分配资源
	
	if (!module->resource) return -2; // 错误处理
	
	module->initalized = 1;
	
	return 0;
}
```

## 析构（死亡）

```
void Module_DeInit(Module_t * module)
{
	if (!module->initialized) return; //检查状态
	
	free(module->resource); //释放资源
	
	module-.resource = NULL;
	
	module->initialized = 0; //重置状态
}
```
